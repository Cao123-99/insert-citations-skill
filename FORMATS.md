# 参考文献格式模板

## GB/T 7714-2015（中国国家标准，中文论文最常用）

### 期刊论文 [J]

```
[序号] 作者1, 作者2, 作者3, 等. 题名[J]. 刊名, 年, 卷(期): 起止页码.

示例：
[1] He K, Zhang X, Ren S, et al. Deep residual learning for image recognition[C]//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition. 2016: 770-778.
```

### 会议论文 [C]

```
[序号] 作者1, 作者2, 等. 题名[C]//会议录名称. 出版地: 出版者, 年: 起止页码.

示例：
[2] Ren S, He K, Girshick R, et al. Faster R-CNN: Towards real-time object detection with region proposal networks[J]. IEEE Transactions on Pattern Analysis and Machine Intelligence, 2017, 39(6): 1137-1149.
```

### 专著 [M]

```
[序号] 作者. 书名[M]. 出版地: 出版者, 年.
```

### 学位论文 [D]

```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年.
```

### 在线资源 [CP/OL] 或 [EB/OL]

```
[序号] 作者. 题名[EB/OL]. [引用日期]. 获取和访问路径.

示例：
[3] Jocher G, Chaurasia A, Qiu J. Ultralytics YOLOv8[CP/OL]. https://github.com/ultralytics/ultralytics, 2023.
```

### 预印本 [J/OL] (arXiv)

```
[序号] 作者1, 作者2, 等. 题名[J/OL]. arXiv preprint, 年份, 编号.

示例：
[4] Ge Z, Liu S, Wang F, et al. YOLOX: Exceeding YOLO series in 2021[J/OL]. arXiv preprint arXiv:2107.08430, 2021.
```

### 专利 [P]

```
[序号] 专利权人. 专利名: 专利号[P]. 公告日期.
```

## IEEE 格式（英文工程论文常用）

### 期刊

```
[1] A. Author, B. Author, and C. Author, "Title of paper," Journal Name, vol. xx, no. xx, pp. xxx-xxx, Year.
```

### 会议

```
[2] A. Author and B. Author, "Title of paper," in Proceedings of Conference Name, Year, pp. xxx-xxx.
```

### 书籍

```
[3] A. Author, Title of Book. City, State: Publisher, Year.
```

---

## APA 7th（社会科学常用）

### 期刊

```
Author, A., & Author, B. (Year). Title of article. Journal Name, Volume(Issue), page-page. https://doi.org/xxxx
```

### 会议

```
Author, A. (Year). Title of paper. In A. Editor (Ed.), Proceedings Title (pp. xx-xx). Publisher.
```

## 格式速查

| 文献类型 | GB/T 7714 标识 | 说明 |
|---------|---------------|------|
| 期刊 | [J] | Journal |
| 会议 | [C] | Conference |
| 专著 | [M] | Monograph |
| 学位论文 | [D] | Dissertation |
| 在线资源 | [EB/OL] | Electronic Bulletin/Online |
| 计算机程序 | [CP/OL] | Computer Program/Online |
| 专利 | [P] | Patent |
| 标准 | [S] | Standard |
| 报告 | [R] | Report |
| 预印本 | [J/OL] | Journal Online（arXiv 等） |

## 规则备忘

1. **作者超过 3 人**：GB/T 7714 用 "等" 或 "et al."；IEEE 用 "et al."
2. **英文作者**：姓在前名在后，名用首字母缩写。`Kaiming He` → `He K`
3. **中文作者**：姓和名之间不加空格
4. **页码**：期刊论文必须有起止页码，会议论文必须有
5. **DOI**：有 DOI 的尽量附上，方便核验
6. **访问日期**：在线资源 [EB/OL] 必须注明引用日期
